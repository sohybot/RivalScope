# 🔥 RivalScope
AI 기반 경쟁 게임 리서치 자동화 도구

> 경쟁 게임 분석을 수 시간 → 수 분으로 단축

---

## 👤 Role
**Product Manager**
- 문제 정의 및 사용자 니즈 분석
- AI 기능 기획 및 UX 설계
- 프로토타입 제작 및 검증

---

## 🎯 Problem

경쟁 게임 리서치는 다음과 같은 문제가 있음:

- 앱스토어 / 커뮤니티 / 뉴스 등 정보가 분산
- 수천 개 리뷰를 수작업으로 분석해야 함
- 담당자 역량에 따라 인사이트 품질 편차 발생

👉 결과  
- 리서치에 수 시간 소요  
- 실시간 대응 어려움

---

## 💡 Solution

RivalScope는  
**게임 이름 1개 입력만으로 경쟁 분석을 자동화**

### 핵심 기능
- 경쟁 게임 프로필 자동 생성
- 유저 반응 (긍정 / 부정) 요약
- 강점 / 약점 비교
- 우리 게임 적용 인사이트 도출

---

## 🧠 How it works

Input → Web Search → AI 분석 → JSON 구조화 → UI 렌더링

- Claude Sonnet 4
- Web Search Tool
- JSON 기반 구조화 출력
